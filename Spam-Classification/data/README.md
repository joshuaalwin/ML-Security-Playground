# Dataset

The SMS Spam Collection dataset is not included in this repository due to size.

## How to Get the Data

The dataset is automatically downloaded when you run the notebook, or manually download from:
- **Source:** [UCI SMS Spam Collection](https://archive.ics.uci.edu/dataset/228/sms+spam+collection)
- **Direct Link:** https://archive.ics.uci.edu/static/public/228/sms+spam+collection.zip

## After Download

Extract to `Spam-Classification/sms_spam_collection/`
