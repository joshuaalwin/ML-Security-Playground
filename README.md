# ML-Security-Playground

A collection of machine learning projects focused on security applications, built while completing the **AI Red Teamer** path on [Hack The Box Academy](https://academy.hackthebox.com/).

## Projects

| Project | Description |
|---------|-------------|
| [Spam-Classification](./Spam-Classification/) | SMS spam detection using Multinomial Naive Bayes with NLP preprocessing |

## Getting Started

Each project has its own `requirements.txt`. To set up a project:

```bash
cd <project-name>
pip install -r requirements.txt
```
